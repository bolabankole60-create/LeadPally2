# Definition of Done

A task is complete only when:

- [ ] It implements the approved frozen architecture.
- [ ] Unit and feature tests are added where appropriate.
- [ ] Existing tests pass.
- [ ] Backend formatting and static checks pass where configured.
- [ ] Frontend lint, type check, and build pass.
- [ ] Documentation is updated where necessary.
- [ ] No critical security issue is introduced.
- [ ] The change is reviewed and merged.
- [ ] The feature is verified in staging when applicable.
